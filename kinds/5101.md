# Input

Clients must provide a URL in the <code>"i"</code> tag field:

# Output

A URL to the generated image.

# Example 1

## Removes background from image specified as input:

```json
{
    "content": "",
    "kind": 5101,
    "tags": [
        [ "i", "https://some-url.net", "url" ]
    ]
}
```
